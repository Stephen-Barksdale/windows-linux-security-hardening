# Linux Hardening Checklist

## Account Review

- Review sudo users
- Disable unused accounts
- Review SSH access
- Confirm password policy

## Logging

- Review /var/log/auth.log
- Review failed SSH attempts
- Confirm log rotation

## Network

- Check listening services
- Confirm firewall status
- Restrict management access

## Patch Management

- Review available updates
- Confirm package versions
- Document exceptions
