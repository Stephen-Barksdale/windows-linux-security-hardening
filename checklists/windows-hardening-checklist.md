# Windows Hardening Checklist

## Account Review

- Review local administrators
- Disable unused local accounts
- Confirm password policy
- Confirm lockout policy

## Logging

- Confirm Security Event Log is enabled
- Review failed logon events
- Review privilege use events
- Review PowerShell logging settings

## Network

- Review Windows Firewall status
- Confirm unnecessary ports are closed
- Confirm RDP access is restricted

## Patch Management

- Confirm latest updates
- Review pending reboot status
- Document exceptions

## Notes

Lab example only.
